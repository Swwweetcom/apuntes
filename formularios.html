Formularios

<form>
Marca un formulario y engloba todos sus campos y botones.

El atributo action indica la acción que se llevará a cabo cuando se envíe el formulario, clicando el botón de submit (normalmente, la acción consiste en enviar el formulario a un servidor).

El atributo method especifica el método HTTP (GET o POST) que se utilizará al enviar el formulario.

<fieldset>
Marca un conjunto de elementos relacionados dentro de un formulario.
<legend>
Sirve como título o leyenda para un <fieldset>.
<label>
Marca la etiqueta de un campo del formulario. Cada <label> está asociada a un <input>.
<input>
Marca un campo de un formulario, donde el usuario puede ejecutar alguna acción, como escribir. El atributo type define de qué tipo de campo se trata. Algunos de los tipos que existen son:
  • <input type="text" />: Marca un campo de texto libre de una línea.
  • <input type="email" />: Marca un campo de e-mail.
  • <input type="submit" />: Marca un botón con el que validar el formulario.
  • <input type="password" />: Marca un campo de contraseña.
  • <input type="email" />: Marca un campo de e-mail.
  • <input type="tel" />: Marca un campo de teléfono.
  • <input type="date" />: Marca un campo donde introducir una fecha.
  • <input type="number" />: Marca un campo numérico.
  • <input type="search" />: Marca un campo de búsqueda.
  • <input type="radio" />: Marca un botón de opción. Los botones de opción permiten al usuario elegir una única opción de entre las posibles.
  • <input type="checkbox" />: Marca un campo de casilla. Las casillas permiten al usuario elegir cero o más opciones de entre las presentes.

El atributo placeholder permite añadir un pequeño texto de ayuda dentro de un input. Este texto de apoyo desaparece en cuanto el usuario empieza a escribir dentro del campo.

El atributo required indica que el campo en cuestión es obligatorio. Si el usuario lo deja vacío, o no lo rellena de manera adecuada, el navegador dará algún tipo de mensaje de error.

<button>
Marca un botón clicable. Con el atributo type="submit" envía el formulario.
<textarea>
Marca un campo de texto de múltiples líneas, a diferencia del <input type="text" />.
<select>
Marca una lista desplegable de opciones, definidas con el elemento <option>.
<option>
Marca cada una de las opciones dentro de un <select>.
