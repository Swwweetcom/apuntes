Selectores

Selector universal *
El selector * se aplica a todos los elementos de un documento. Dependiendo de cómo lo usemos, puede llegar a afectar un poco al rendimiento de la página.
Selector de elemento
Para aplicar estilos a un elemento (por ejemplo, a todos los párrafos <p>), se ultiliza el selector de elemento.
Ejemplo:
  p {
color: black;
}
Selector de id
Para aplicar estilos a un id (por ejemplo <div id="importante">), se ultiliza el selector de id #.
Ejemplo:
  #importante {
color: red;
}
Selector de clase
Para aplicar estilos a un class (por ejemplo <div class="frecuente">), se ultiliza el selector de clase ..
Ejemplo:
  .frecuente {
color: blue;
}
Selector de descendiente
Para aplicar estilos a un elemento que es hijo de otro, es decir, que está debajo en el árbol del documento, no importa cuántos niveles por debajo.
Ejemplo
  <div id="importante">
<p>Un párrafo con texto <span>destacado</span>.</p>
</div>
 
  #importante span {
color: red;
}
Selector de hijo directo
Para aplicar estilos a un elemento que es hijo directo de otro, es decir, que está inmediamente debajo en el árbol del documento, se utiliza el selector de hijo >.
Ejemplo
  <div id="importante">
<p>Un párrafo</p>
</div>
 
  #importante > p {
color: red;
}

Pseudo-clases

:hover
Indica el estado de un elemento (normalmente, un link) cuando pasamos el ratón por encima. Ejemplo: a:hover{color: blue;}
:active
Indica el estado de un elemento (normalmente, un link o un botón) cuando lo presionamos. Ejemplo: a:active{border: none;}
:focus
Indica el estado de un elemento accionable (normalmente, un link o un botón) cuando tiene el foco (el efecto al navegar con el tabulador entre links). Ejemplo: a:focus{outline: 2px dashed blue;}
:visited
Indica un link que ya ha sido visitado. Ejemplo: a:visited{color: grey;}
:first-child
Se refiere al primer elemento dentro de un contenedor en concreto. Ejemplo: .articulo p:first-child{font-size: 20px;}
:last-child
Se refiere al último elemento dentro de un contenedor en concreto. Ejemplo: .listado li:last-child{border-bottom: none;}
:nth-child()

Se usa para seleccionar un elemento concreto en una lista de varios.

Si indicamos un número (ejemplo: p:nth-child(3), nos referimos a ese número de elemento en particular. Si añadimos la letra n (ejemplo: li:nth-child(3n)), indicamos repetición, cada "n" elementos.

Si queremos usar :nth-child con repetición, pero queremos que empiece a contar desde un elemento distinto al primero, podemos modificarlo sumando o restando números. Ejemplo: li:nth-child(4n+2).

Pseudo-elementos

::before y ::after

Permiten añadir contenido normalmente decorativo desde CSS, sean imágenes, texto o un espacio vacío, delante (si usamos before) o detrás (si usamos after) del elemento que queramos. Para declarar el contenido en before o after usaremos la propiedad content. Ejemplo: p::before{content: "·"}.

Es habitual ver escrito :before y :after, con solo ":" en lugar de "::". Escrito de esta manera, aunque no es la correcta, se soporta también en Internet Explorer 8.

Si queremos sencillamente generar un "espacio fantasma" al que daremos estilos, podemos crearlo declarando el content vacío, por ejemplo: 

  p::before{
content: "";
display: inline-block;
background: hotpink;
width: 20px;
height: 20px;
}

Modelo de caja

width
Determina la anchura de un elemento. Ejemplo: width: 100px;.
height
Determina la altura de un elemento. Ejemplo: height: 100px;.
min-width / max-width
Determinan la anchura mínima o máxima de un elemento.
min-height / max-height
Determinan la altura mínima o máxima de un elemento.
margin
Determina los márgenes de un elemento. Se puede declarar un margen concreto (margin-top / margin-right / margin-bottom / margin-left), o añadir los cuatro márgenes a la vez, en el orden arriba-derecha-abajo-izquerda: margin: 5px 10px 5px 20px.
padding
Determina el espacio entre el límite de un elemento y su contenido. Se puede declarar un padding concreto (padding-top / padding-right / padding-bottom / padding-left), o añadir los cuatro a la vez, en el orden arriba-derecha-abajo-izquerda: padding: 5px 10px 5px 20px.
box-sizing
Determina el modelo de caja que se aplica al elemento. El modelo de caja por defecto es content-box, en el cual la anchura de un elemento es igual al width + padding + border (con la altura ocurre lo mismo). Si declaramos el modelo de caja border-box, la anchura total será la que marquemos en el width. Para resetear el modelo de caja en todo el documento, usa este código al principio de tu CSS:
* {box-sizing: border-box;}

Bordes

border-width
Determina la anchura de un borde: border-width: 3px;.
border-color
Determina el color de un borde: border-color: yellow;.
border-style
Determina el estilo de un borde: border-style: solid;. solid es una línea sólida, dotted es una línea de puntos, dashed es una línea discontínua y double es una línea doble.
border
Los estilos de borde se pueden aplicar bajo una única declaración "shorthand": primero la anchura, después el estilo y por último el color. Ejemplo: boder: 3px solid yellow;.
border-radius
Determina el radio de redondeado de las esquinas de un elemento. Se puede dar el mismo radio a todas las esquinas: boder-radius: 5px; o uno diferente a cada una: border-radius: 5px 10px 5px 0;. En este último caso, el orden en que se aplica es esquina superior izquierda, superior derecha, inferior derecha, inferior izquierda.

Layout

float
Hace que un elemento de bloque se coloque (flote) junto a otro. Se puede usar tanto float: left; para flotar a la izquierda, como float: right; para flotar a la derecha.
overflow
Con la propiedad overflow indicamos qué tiene que hacer un contenedor si su contenido se sale de su caja. Usando overflow: hidden; le diremos que oculte lo que se salga fuera; también es una buena forma de forzar a un contenedor a cubrir completamente a su contenido, cuando este contenido está flotando.
clear
Sirve para despejar un elemento que tiene otros elementos flotando a su lado. Se puede usar clear: left; para despejar el lado izquierdo, clear: right; para el derecho, o clear: both; para ambos a la vez.
clearfix
Es la técnica más popular para limpiar los floats. Para usarla, tenemos que añadir al bloque que contenga elementos que flotan esta CSS: 
  .contenedor:before,
.contenedor:after {
content: " ";
display: table;
}

.contenedor:after {
clear: both;
}
display

Esta propiedad cambia la forma en la que un elemento se renderiza en la página: como un bloque (display:block;), como una línea (display: inline;), o como una mezcla (display: inline-block). Los elementos con display: inline-block; no provocan un salto de línea, pero a diferencia de los display: inline; aceptan propiedades como margin-top, margin-bottom o height.

Si queremos ocultar un elemento, podemos usar display: none;

position

La propiedad position nos permite “sacar” a un elemento del flujo normal de la página. Con position: relative; movemos un elemento respecto a su posición original. Con position: absolute; lo movemos absolutamente respecto a la ventana del navegador.

Para que un elemento posicionado absolutamente tenga como punto de partida un contenedor en particular en vez de la ventana del navegador, tendremos que aplicar position: relative; a dicho elemento.

Para desplazar un elemento posicionado relativamente o absolutamente, usaremos las propiedades top, bottom, left o right, generalmente solo una de cada eje (declarar top y bottom a la vez, por ejemplo, no siempre dará el resulado que esperamos).

Cuando dos elementos posicionados relativa o absolutamente se solapan, podemos usar la propiedad z-index para decidir cuál queda encima de cuál. Ejemplo: z-index: 1;

calc()

La función calc() nos permite hacer cálculos sencillos (sumas, restas, multiplicaciones, divisiones) de valores. Aunque está bastante limitado, nos permite hacer cosas como asignar una anchura a un elemento de un 100% (u otro porcentaje) menos una cantidad en píxeles. Ejemplo: width: calc(100% - 20px);

Para que calc() funcione correctamente, separa los signos aritméticos con espacios.

Background

background-color
Sirve para poner un color de fondo a un elemento. Por ejemplo, background-color: #CCCCCC;
background-image
Sirve para poner una imagen de fondo a un elemento. Por ejemplo, background-image: url(images/bg.jpg);
background-position

Sirve para posicionar una imagen de fondo en un elemento. Los fondos se deben posicionar horizontalmente (left/center/right) y verticalmente (top/center/bottom). También se pueden posicionar una cantidad fija de píxeles, o un porcentaje. Por defecto, las imágenes de fondo se posicionan en left top. Por ejemplo, background-position: right center;.

También podemos especificar una cantidad fija de píxeles o un porcentaje desde una esquina diferente a la por defecto, combinando palabras y unidades: background-position: right 10px bottom 20px;

background-repeat
Sirve para indicar la repetición de una imagen de fondo. repeat repite la imagen tantas veces como sea posible. repeat-x repite la imagen verticalmente tantas veces como sea posible. repeat-y repite la imagen horizontalmente tantas veces como sea posible. no-repeat impide que la imagen se repita. El estado por defecto de las imágenes de fondo es repeat. Por ejemplo, background-repeat: no-repeat;.
background
Los estilos de fondo se pueden aplicar bajo una única declaración "shorthand": primero la imagen, después la posición, después la repetición y por último el color. Ejemplo: background: url(images/bg.jpg) right center no-repeat #CCCCCC;.

Color

Nombre
Algunos colores, como black, white, red o blue se pueden utilizar simplemente con su nombre. Aquí encontrarás los nombres de los 147 colores que existen en CSS: http://www.colors.commutercreative.com/grid/
Hexadecimal
Los colores se pueden utilizar mediante su código hexadecimal, como por ejemplo #000000, #ff0000 o #dadada.
rgba
Los colores se pueden utilizar indicando sus valores en el canal rojo, verde, azul y alfa. Este último canal indica la cantidad de transparencia que tendrá el color, y se expresa en base 1. Ejemplo: background-color: rgba(255, 0, 0, .5);.
hsla
Los colores se pueden utilizar indicando sus valores de tono, saturación, luminosidad y alfa. El tono indica el punto en el que se encuentra el color en la rueda de color, y se expresa con un valor (grado) comprendido entre 0 y 360. La saturación se expresa con un porcentaje, donde 0% es gris y 100% es el color en su máxima saturación. La luminosidad se exprea con un porcentaje, donde 0% es negro, 50% es el color puro y 100% es blanco. El canal alfa indica la cantidad de transparencia que tendrá el color, y se expresa en base 1. Ejemplo: background-color: hsla(310, 100%, 50%, .8);. Esta herramienta te puede ayudar con este tipo de colores: http://hslpicker.com.

Degradados

linear-gradient

Un degradado lineal es la forma más habitual de degradado (de un punto a otro siguiendo una línea recta y con dos o más colores). Todos los degradados se tratan como imágenes de fondo, y se declaran en la propiedad background-image o background.

La forma de escribirnos es declarando el ángulo (en grados) o la dirección y los diferentes colores de corte.

Ejemplo de degradado entre dos colores de arriba abajo:
div{background-image: linear-gradient(180deg, yellow, blue);}

Ejemplo de degradado entre varios colores, en diagonal:
div{background-image: linear-gradient(to bottom right, yellow 0%, blue 10%, red 50%, purple 100%);}

radial-gradient

Un degradado radial o en forma cicular se declara indicando el tamaño (si queremos), forma (circle o ellipse), posición del vértice, cortes de color.

Ejemplo de degradado circular posicionado en la esquina superior derecha entre dos colores:
background-image: radial-gradient(circle at right top, yellow, blue);

Ejemplo de degradado elíptico centrado entre 4 colores:
background-image: radial-gradient(ellipse at center, yellow 10%, blue 20%, green 50%, brown 100%);

repeating-linear-gradient

Este tipo de degradado nos permite crear un patrón o degradado que se repite de forma continua. Ejemplo: repeating-linear-gradient(180deg, #1ac6cc, #1ac6cc 7%, #646464 10%).

La forma de que tenga espacio para repetirse es no declarar el 100% del espacio que ocuparán los colores (en el ejemplo anterior, tenemos hasta el 10%), o bien usar unidades absolutas (píxeles).

Sombras

box-shadow

La forma de declarar sombras en un elemento es con la propiedad box-shadow. Los valores que se introducen son el desplazamiento de la sombra en el eje x, el desplazamiento en el eje y, la difuminación de la sombra, el spread (un espacio de color sólido antes de que empiece el difuminado; es opcional) y el color de la sombra. Además, si usamos la propiedad inset, la sombra se renderizará por dentro del elemento en vez de por fuera.

Ejemplo de sombra normal:
box-shadow: 5px 5px 3px rgba(0, 0, 0, .4);

Ejemplo de sombra con spread:
box-shadow: 5px 5px 3px 2px rgba(0, 0, 0, .4);

Ejemplo de sombra con inset:
box-shadow: inset 5px 5px 3px rgba(0, 0, 0, .4);

Un elemento puede tener más de una sombra, que se posicionan una debajo de la otra. Ejemplo:
box-shadow: 0 5px 0 lightblue, 0 10px 0 hotpink;

text-shadow

La forma de declarar una sombra de texto es igual que la de box-shadow, con la única diferencia de que en text-shadow no existe el inset (no se puede renderizar la sombra por dentro del texto).

Ejemplo: text-shadow: 2px 2px 3px rgba(0, 0, 0, .4);

Texto

font-family
Sirve para indicar la familia tipográfica de un elemento de texto. Se puede indicar más de una familia, y el navegador escogerá la primera que encuentre disponible. Si el nombre de la familia tipográfica tiene más de una palabra, deberá ir entre comillas. Ejemplo: font-family: "Helvetica Neue", Helvetica, Arial, sans-serif;
color
Sirve para indicar el color de un elemento de texto (párrafo, enlace, etc.). Ejemplo: color: red;.
font-weight
Permite cargar un peso distinto de la tipografía (bold, normal, 300, 600, etc).
Ejemplo: font-weight: bold;
font-style
Indica si el texto se renderizará con la tipografía normal (font-style: normal;), cursiva (font-style: italic;) u oblícua (font-style: oblique;).
text-align
Sirve para indicar la alineación del texto, que puede ser left (alineado a la izquierda, opción por defecto), right (alineado a la derecha), center (centrado) o justify (justificado). Ejemplo: text-align: center;.
text-decoration
Indica si un elemento va subrayado (text-decoration: underline;), tachado (text-decoration: line-through;) o sobrerayado (text-decoration: overline;). Si queremos quitarle cualquiera de estos efectos, usaremos text-decoration: none;
vertical-align
Indica la posición vertical de un elemento de línea (un texto o una imagen) respecto a otro. Podemos indicar que los elementos se alineen por la línea base (vertical-align: baseline;), centrado (vertical-align: middle;), u otros valores respecto a la posición y altura del texto: super, sub, text-top, text-bottom, top, bottom. También con unidades, por ejemplo vertical-align: 4px;
line-height
Señala la altura de la línea (de ello depende en parte el espacio que deja una línea por arriba y por abajo). Esta altura de línea se puede especificar en una unidad (ejemplo: line-height: 24px;) o dejar que sea proporcional al tamaño del texto (ejemplo: line-height: 1.5;).
text-indent
Permite que el texto que nos interese tenga un margen inicial en la línea. Se puede usar para que la primera línea de los párrafos de un texto esté indentada (tenga un pequeño margen), por ejemplo: p{text-indent: 20px;}.

Listas

list-style

list-style es el método abreviado para los estilos de lista: indicamos el tipo de lista, si tiene una imagen como bullet y la posición de los bullets o números respecto a la lista. Ejemplo: list-style: square inside;

Si queremos que una lista no tenga bullets o numeración, usaremos list-style: none;.

list-style-type
Indica el tipo de bullet o numeración en una lista. Por defecto, en un ul es list-style-type: disc;. Otros valores son circle, square, lower-latin, etc…
list-style-position
Indica la posición de los bullets o números respecto a su contenedor (la lista). Puede ser inside (dentro de la lista) u outside (fuera de la lista).
list-style-image
Permite asignar una imagen como bullet personalizado en una lista. Ejemplo: list-style-image: url(image.gif);

Columnas

column-count

Determina el número de columnas que tiene un bloque de texto (por defecto es 1). Ejemplo: column-count: 3;

column-gap

Determina el espacio entre las distintas columnas. Se puede indicar en píxeles, ems... Ejemplo: column-gap: 50px;

column-rule

Determina los estilos para una línea de separación entre las columnas. Se declara de manera similar a los bordes: primero el ancho (que puede ser thin, medium, thick, o de un grosor específico), después el estilo (como en los bordes: solid, dashed, dotted o double) y, por último, el color. Ejemplo: column-rule: 1px solid black;

Webfonts

@font-face

Esta declaración sirve para poder utilizar tipografías distintas a las instaladas en el sistema del usuario. Las tipografías deben estar alojadas en el servidor (por ejemplo, en una carpeta "fonts"). Para poderlas usar, debemos darle un nombre a la familia tipográfica, especificar la ruta donde están los archivos de las fuentes (en sus distintos formatos, si fuera necesario) y, si queremos, indicar el estilo de la fuente (bold, italic, etc). Un ejemplo de uso sería:

  @font-face {
font-family: "Museo";
src: url('/fonts/museo.woff') format('woff');
font-weight: normal;
}

Para poder usar esta tipografía en nuestra página, simplemente tendríamos que llamarla con el nombre que le hayamos dado, como por ejemplo:

  body {
font-family: "Museo", Verdana, sans-serif;
}
Formatos de fuentes

Existen múltiples formatos de fuentes. Los estándar y mejor optimizados son los formatos Woff y Woff2, soportados en los navegadores modernos. Si queremos, podemos cargar también el formato truetype (ttf), soportado en la mayoría de navegadores más antiguos. Si queremos utilizar más de un formato para garantizar un mayor soporte, se hace de la siguiente forma: 

  @font-face {
font-family: "Museo";
src:
  url('/fonts/museo.woff2') format('woff2'),
  url('/fonts/museo.woff') format('woff'),
  url('/fonts/museo.ttf') format('truetype');
}

Para convertir un archivo tipográfico a múltiples formatos, podemos utilizar el generador de webfonts de FontSquirrel.

Servicios de webfonts

Existen múltiples servicios, tanto gratuitos como de pago, desde los que podemos utilizar webfonts. Algunos de los más destacados son:

  • Google Fonts: servicio gratuito de Google. Para poder usar sus tipografías, solo hay que seleccionaras, añadirlas a una colección, seleccionar los pesos que queremos y añadir el código que nos muestra al <head> de nuestro documento HTML. En este artículo podemos encontrar una lista de algunas de las mejores tipografías disponibles en Google Fonts, entre otros.
  • Typekit: servicio de pago de Adobe en el que, mediante una subscripción anual, tenemos acceso a una enorme librería de tipografías comerciales.
  • Myfonts: página donde podemos comprar licencias de una enorme variedad de fuentes. Una vez compradas, se pueden descargar y utilizar en nuestras páginas mediante @font-face.

Media queries

@media

Las reglas @media sirven para englobar entre llaves una serie de estilos que solo se emplearán si la web se reproduce en un medio concreto, como en pantalla (@media screen{/*CSS para pantalla*/}) o impresión (@media print{/*CSS para impresión*/}).

@media queries

Las queries (condiciones) son una extensión de las @media, que típicamente nos permitirán añadir condiciones de estilo en función de la anchura y altura del navegador. La forma normal de escribirlas es: 

@media screen and (/* condición de altura o anchura*/){
/* estilos para esa condición */
}

Las condiciones más normales serán anchura mínima (min-width) y anchura máxima (max-width).

La forma lógica de escribir las media queries es debajo de la CSS que quieren sobreescribir, y en un orden creciente o decreciente en función de nuestra estrategia. Si hemos empezado nuestra web por el layout de escritorio, normalmente escribiremos nuestras media queries así: 

  .container{
/*CSS inicial: tres columnas de contenido*/
}

@media screen and (max-width: 900px){
.container{
  /*CSS para dos columnas de contenido*/
}
}

@media screen and (max-width: 650px){
.container{
  /*CSS para una columna de contenido*/
}
}

Si hemos empezado por el layout de móvil, el orden normal de nuestras media quieres será de menor a mayor, y usando min-width: 

  .container{
/*CSS inicial: una columna de contenido*/
}

@media screen and (min-width: 650px){
.container{
  /*CSS para dos columnas de contenido*/
}
}

@media screen and (min-width: 900px){
.container{
  /*CSS para tres columnas de contenido*/
}
}

combinar condiciones

Las condiciones de una media query se pueden combinar, especialmente cuando nos interesa que unos estilos se apliquen solo entre un tamaño y otro, y no tener que sobreescribirlos en otra media query. Para hacer esto, sencillamente añadimos la palabra and y otra condición entre paréntesis, por ejemplo: @media screen and (min-width: 400px) and (max-width: 800px){/*estilos que solo se aplicarán entre 400px y 800px de ancho*/}.

Transiciones

transition
Para añadir una transición a un elemento en un cambio de estado (por ejemplo, :hover o cuando se le añade o quita una clase), usaremos transition, declarando el tiempo que durará la transición en segundos o milisegundos y la propiedad a transicionar (por defecto, all).
Ejemplo: a{transition: .4s background-color;}

Transformaciones

transform
Permite modificar el aspecto o posición de un elemento sin alterar su entorno (por ejemplo, hacer algo más grande sin mover los elementos de alrededor). Los tipos de transformación habituales son translate(), scale(), rotate(), skewX() y skewY().
transform: translate()
Desplaza a un elemento en el eje X y el eje Y. Ejemplo: 
  .ejemplo{
transform: translate(10px, 20px);
}
transform: scale()
Altera el tamaño de un elemento en base a 1. Ejemplo: 
  .ejemplo{
transform: scale(1.5);
}
transform: rotate()
Gira un elemento a partir de su centro, expresado en grados (deg). Ejemplo: 
  .ejemplo{
transform: rotate(15deg);
}
transform: skewX()
Deforma un elemento sobre el eje X, expresado en grados (deg). Ejemplo: 
  .ejemplo{
transform: skewX(15deg);
}
transform: skewY()
Deforma un elemento sobre el eje Y, expresado en grados (deg). Ejemplo: 
  .ejemplo{
transform: SkewY(15deg);
}

Animaciones

Animaciones CSS

Las animaciones permiten hacer movimientos y transiciones sencillas en elementos web, sin necesidad de interacción por parte del usuario.

Las animaciones se definen con @keyframes y se asignan posteriormente a los elementos que nos interese animar.

Algunos navegadores pueden necesitar todavía que declares las animaciones con su prefijo (-webkit, -moz, etc). Compruébalo en Can I Use.

@keyframes

Define los pasos que seguirá una animación, y le da un nombre para poderse referir a ella más adelante.

Los estilos de cada paso de la animación se engloban entre {}. Los pasos pueden indicarse con from y to, si son solo dos, o con %.

Ejemplo: 

  @keyframes ejemplo{
0%{
  width: 20px;
}
20%{
  width: 50px;
}
80%{
  width: 90px;
}
100%{
  width: 240px;
}
}

animation

Esta propiedad permite asignar una animación a un elemento, indicando en este orden: nombre de la animación, duración en segundos (s) o milisegundos (ms), velocidad (linear, ease-in, ease-out, ease-in-out), retraso en empezar, repeticiones, dirección en la que se produce (normal, reverse, o alternate si queremos que cambie con cada repetición). Tienes algunas opciones más en la Mozilla Developer Network.

Un ejemplo de asignación de animación: 

  .elemento{
animation: ejemplo 3s ease-out 3s 3 alternate;
}
