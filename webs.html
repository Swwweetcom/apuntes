Webs de referencia

  • Mozilla Developer Network, un montón de explicaciones sobre características de HTML, CSS y JavaScript.
  • CSS Tricks, un blog muy activo con muchos recursos de CSS y HTML.
  • Can I Use, compara el soporte de diferentes tecnologías (HTML, CSS, JS) en los navegadores en sus versiones actuales, pasadas y futuras.
  • jQuery.com, la web oficial de jQuery.
  • jQAPI, una web con documentación alternativa sobre jQuery.
