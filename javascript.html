Principios

¿Para qué sirve JavaScript?

JavaScript es el lenguaje de programación que se ejecuta del lado del cliente (es decir, en el navegador) en vez de en el servidor, como es el caso de PHP.

Las funciones de JavaScript son: acceder al contenido de una web, modificar el contenido de una web, programar reglas y reaccionar a eventos.

La programación del lado del navegador puede sufrir muchos problemas (el archivo de JS no se descarga bien, sufre modificaciones en el navegador del usuario, o una función estaba mal escrita y el script deja de funcionar), por lo que se recomienda siempre que una web sea capaz de emplearse si JavaScript(aunque sea de manera muy simplificada).

Scripts
Los scripts son instrucciones que se ejecutan de manera lineal, como los pasos de una receta. Es la forma en la que escribimos instrucciones en JavaScript.
Programación orientada a objetos
Los lenguajes de programación modernos tienden a usar un modelo de datos que se llaman "objetos". Estos objetos son elementos que tienen propiedades y valores propios, y también métodos (funciones internas). Una web se renderiza en un navegador como el objeto document, que se visualiza dentro del objeto window o ventana del navegador.
Etiqueta <script>

La etiqueta script se escribe <script></script>, y dentro podemos poner cualquier código JS, que se ejecutará dentro de la página. Si queremos cargar un archivo externo .js, utilizaremos el atributo src dentro de esta etiqueta. Ejemplo: <script src="scripts.js"></script>

El código JavaScript se ejecuta en el punto del HTML en el que se encuentra, por lo que se suele recomendar ponerlo al final de nuestro html (justo antes del cierre de <body>), para que no bloquee la carga de la página.

Sintaxis

Las declaraciones en JavaScript (por ejemplo, asignar un valor a una variable) se finalizan con ;, de forma muy similar a CSS. Ejemplo: var edadUsuario = 20;

Los comentarios en JavaScript se escriben igual que en CSS: /* aquí el comentario */. Además, en JS contamos con el comentario de línea, poniendo // al principio de la misma.

Cuando queremos agrupar una o varias declaraciones en una función o un condicional (como if…else), usaremos {}. Ejemplo: if(edadUsuario>20){alert("Eres mayor de edad.");}

El paréntesis suele indicar una función, y lo que ponemos entre paréntesis, el parámetro que pasamos a dicha función. Ejemplo: alert("Hola");

El punto (.) indica pertenencia, por lo que cuando lo encontremos en una declaración, probablemente se refiere a la propiedad de un objeto. Ejemplo: document.URL;. Si lo que tenemos a continuación es un paréntesis, probablemente se refiere a un método (función interna) propio del objeto. Ejemplo: console.log("Hola");

En JavaScript, los listados de elementos se empiezan a contar desde el 0, no el 1. Si nos queremos referir al primero, hablaremos por tanto del "elemento en la posicion 0 del listado".

DOM
El DOM (Document Object Model) es el objeto document, que es como se renderiza una página web en el navegador. Cuando hablamos de que usamos JavaScript para recoger o alterar información de una web, hablamos siempre de "acceder al DOM".

Consola

Consola
La consola es una pestaña en el inspector de DOM (o las herramientas de desarrollo) que nos permite introducir instrucciones de JS y leer mensajes de error o de log. Para mostrar mensajes en la consola, podemos utilizar el método log del objeto console, de esta manera: console.log("mensaje que quieras mostrar");

Variables

Variables

Las variables son elementos que pueden tener un valor que se pueda modificar a lo largo de nuestro script, sea este texto, un número, un valor booleano o un array.

Una variable se declara con la instrucción var y el nombre que queramos (que no empiece por números, tenga guiones o espacios, o sea una palabra restringida de JavaScript). Ejemplo: var edadUsuario;

Si sabemos el valor inicial de una variable, podemos asignarlo al mismo tiempo que la declaramos: var edadUsuario = 20;

Además, podemos hacer una declaración múltiple de variables separadas por comas.
Ejemplo: var edadUsuario = 20, nombreUsuario = "Laura", socio = true;

Strings (texto)

El valor de las variables cuyo valor es texto se declara entre comillas, lo que nos permite poner casi cualquier carácter sin que el resto de nuestro script se vea afectado. Ejemplo: var nombreUsuario = "Laura López";

Cuando queremos poner comillas dentro de comillas (por ejemplo, un string que contenga etiquetas HTML), deberíamos anidar comillas simples dentro de las comillas dobles.
Ejemplo: var respuestaForm = "<p class='respuesta' >¡Muchas gracias!</p>";

Variables numéricas
Un valor numérico se asigna a una variable sin comillas (de lo contrario, estaríamos creando un string). Ejemplo: var edadUsuario = 20;
Variables booleanas
Una variable booleana es aquella cuyo valor es "verdadero" (true) o "falso" (false).
Ejemplo: var esSocio = true;
Arrays (listados)

Un array es una variable especial que puede contener varios valores (o un objeto bastante simple).

La forma más sencilla de declarar un array es con [], es decir: var coloresDisponibles = ["rojo", "azul", "amarillo"];

La forma más sencilla de recuperar un valor concreto de un array es refiriéndose a la posición que ocupa en el listado. Ejemplo: coloresDisponibles[0];

Operadores

Operador de asignación: =
El operador = permite asignar un valor a una variable. Para comparar dos valores, usaremos el comparador ==.
Operador aritmético de adición: +
El operador + permite sumar valores numéricos o concatenar strings. Es el único operador aritmético que podremos usar con texto.
Operador aritmético de sustracción: -
Permite hacer una resta entre dos valores numéricos.
Operador aritmético de división: /
Divide dos valores numéricos.
Operador aritmético de multiplicación: *
Multiplica dos valores numéricos.
Operador aritmético de incremento: ++
Podemos añadir el operador ++ al final del nombre de una variable para incrementar en 1 su valor. Una variable "ejemplo" que tenga un valor de 3, subirá a 4 si escribimos ejemplo++;.
Operador aritmético de decremento: --
Podemos añadir el operador -- al final del nombre de una variable para reducir en 1 su valor. Una variable "ejemplo" que tenga un valor de 3, bajará a 2 si escribimos ejemplo--;.
Operador aritmético de módulo: %
Permite obtener el resto de una división de dos números (por ejemplo, 10 % 3 daría como resultado 1).
Operador de comparación: ==
Permite comprobar si dos valores son iguales.
Operador de comparación: !=
Permite comprobar si dos valores son diferentes.
Operador de comparación: ===
Podemos usar === en vez de == si queremos comprobar no solo que dos valores son iguales, sino que son estrictamente del mismo tipo (por ejemplo, no es lo mismo el valor numérico 3 que el valor literal "3", pero == diría que sí es lo mismo, mientras que === no).
Operador de comparación: !==
Al igual que ===, permite comparar dos valores estrictamente; en este caso si el valor y/o el tipo de variable son diferentes.
Operador de comparación: >
Permite saber si un valor es mayor que otro.
Operador de comparación: <
Permite saber si un valor es menor que otro.
Operador de comparación: >=
Permite saber si un valor es mayor o igual que otro.
Operador de comparación: <=
Permite saber si un valor es menor o igual que otro.
Operador lógico AND: &&
Permite saber si dos expresiones se cumplen. Por ejemplo: ((5 > 2) && (2 <= 3))
Operador lógico OR: ||
Permite saber si al menos una de dos expresiones se cumplen. Por ejemplo: ((4 > 2) || (1 > 6))
Operador lógico NOT: !
Permite saber si una expresión no se cumple. Por ejemplo: !(4 < 2)

Seleccionar nodos en el DOM

document.getElementById()
Una forma de seleccionar un nodo (elemento) en el HTML para poder manipularlo, es por su atributo id. Por ejemplo, si queremos acceder al div con el id="principal" y guardar esta búsqueda en una variable, podemos escribir var contenedor = document.getElementById("principal");
document.querySelector()
Devuelve el primer elemento que encuentra con el selector CSS que hayamos utilizado. En este caso, al contrario que con otros métodos como document.getElementById(), tendremos que especificar si es un id con una #, una clase con un . o un tag html. Ejemplo: document.querySelector(".visible");
document.querySelectorAll()
Similar a document.querySelector(), pero devuelve un listado de nodos (array) con todos los elementos que coincidan con la selección, por ejemplo: var listado = document.querySelectorAll("li");… aunque solo exista un elemento de este tipo. Por eso, para referirnos a un elemento de esa lista de nodos, nos referiremos a su posición en el array: listado[0];

Alterar el contenido del DOM

elemento.textContent
Hay varias formas de alterar el contenido de un nodo. Si solo queremos cambiar el texto de un nodo, podemos hacerlo accediendo a la propiedad textContent. Por ejemplo, para un nodo que hayamos guardado en la variable "contenedor", podemos cambiar su texto así: contenedor.textContent = "Un nuevo texto";
document.write()
Una forma de escribir contenido con JS en el HTML es document.write(). Sin embargo, este código ejecuta la escritura en donde la encuentre, por lo que normalmente no lo pondremos en un archivo externo sino directamente en el HTML. En general, no es aconsejable usar este método para añadir contenido al DOM.

Alterar clases en elementos del DOM

elemento.classList.add()
Permite añadir una clase a un elemento del DOM. Ejemplo: elemento.classList.add("activo");
elemento.classList.remove()
Permite quitar una clase a un elemento del DOM. Ejemplo: elemento.classList.remove("activo");
elemento.classList.contains()
Permite comprobar si un elemento del DOM tiene una clase en particular. Ejemplo:
if (elemento.classList.contains("activo")){/*código*/}
elemento.classList.toggle()
Permite añadir una clase a un elemento del DOM si no la tiene, o quitársela si la tiene. Ejemplo: elemento.classList.toggle("activo");

Alterar atributos en elementos del DOM

elemento.getAttribute()
Permite recuperar el valor de un atributo de un elemento. Ejemplo:
var valorHref = elemento.getAttribute("href");
elemento.setAttribute()
Asigna el valor de un atributo en un elemento, y el atributo si no existiera. Ejemplo:
elemento.setAttribute("href", "http://google.com");
elemento.removeAttribute()
Elimina un atributo de un elemento. Ejemplo: elemento.removeAttribute("style");
elemento.hasAttribute()
Comprueba si un elemento tiene un atributo o no. Ejemplo:
if(elemento.hasAttribute("href")){/* código */}

Condicionales

if… else

La forma más habitual de que unas líneas de nuestro script se ejecuten solo cuando se cumpla una condición es con if. Escribimos if, la condición entre paréntesis y después, entre {}, las líneas de código sujetas a la condición.

Podemos encadenar condiciones con else. Tras el cierre de {} del primer if, podemos escribir una alternativa por si no se cumple la primera condición.

Si queremos que la segunda opción o posteriores solo se apliquen si se da una condición, usaremos else if, seguido de la condición entre paréntesis y luego las líneas de código entre {}.

Ejemplo de todo esto: 

var edadUsuario = 18;

if(edadUsuario > 17){
console.log("es mayor de edad");
}
else if(edadUsuario < 12){
console.log("es un niño");
}
else{
console.log("es un adolescente");
}

Loops

for
Un loop como for permite repetir varias veces una serie de instrucciones empleando para ello un contador. De esta manera, podemos decirle que empezando de 0, y hasta que no queden más nodos en un array, repita una instrucción, sumando 1 al contador cada vez. Ejemplo de for en funcionamiento: 
for(var contador=0; contador < listaTabs.length; contador++){
 listaTabs[contador].classList.add("visible");
}

Funciones

function()
Una función nos permite agrupar código entre {} para poderlo reutilizar o llamar cuando es necesario. Normalmente le asignaremos un nombre y podemos pasarle argumentos a través del paréntesis. Ejemplo: 
 function escribeConsola(mensaje){
  console.log(mensaje);
}

//En otra parte del código, la llamamos
escribeConsola("Hola!");

Eventos

Los eventos

Los eventos son cambios en el DOM, provocados por el ratón (por ejemplo, click o mouseover), por el navegador (load o scroll), teclado (keypress), foco (focus o blur), formularios (submit o change)…

En JavaScript, muchas veces recurriremos a estos eventos para programar reglas (por ejemplo, que se validen los campos de un formuario antes de enviarse, que es el evento submit). Tienes una lista completa en la Mozilla Developer Network.

evento.preventDefault()
Este método evita que los elementos sobre los que se ha hecho clic realicen su comportamiento por defecto, como enviar un formulario cuando se ha hecho clic en un botón con type="submit" o acceder a una url cuando se ha hecho clic en un link.
evento.stopPropagation()
Este método evita el "event bubling", o que un clic en un elemento que ya tenía una función asociada (mendiante un listener) afecte a un elemento padre que también tenga una función asociada.
return false;
Esta expresión lanza a la vez evento.preventDefault() y evento.stopPropagation(), pero también deja de procesar el código de una función que pueda venir a continuación, por lo que muchos desarrolladores prefieren evitar su uso.
evento.target
Si asociamos un evento (por ejemplo, el evento click) con una función, es posible que desde esta función nos queramos referir a este evento, por lo que muchas veces pasamos el propio evento como la primera variable (ejemplo: function(evento){ /*código*/ }). Este evento es un objeto que contiene una serie de propiedades, de las cuales la más popular es evento.target, que indica el nodo (tag html) en el que se ha hecho click (o equivalente).

Listeners

Los listeners
Un event listener nos permite asociar funciones a eventos concretos tanto del navegador como nodos en el DOM. Debemos procurar no asignar el mismo más de una vez en cada elemento o provocaremos que una función se ejecute varias veces.
elemento.addEventListener()
Para asignar un listener, seleccionaremos un nodo y mediante addEventListener() le asociaremos un evento y una función, y opcionalmente false o true si queremos asegurar la dirección del "event bubling" (por defecto false).
Ejemplo: 
function comprobarFormulario(){
  //código de la función
}
var formulario = document.getElementById("formulario-envio");
formulario.addEventListener("submit", comprobarFormulario, false);
